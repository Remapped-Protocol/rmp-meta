# Contributing to Remapped Protocol

Thank you for your interest in Remapped Protocol (RMP).

At this stage, the project is in an early documentation and design phase.
Not all repositories accept code contributions yet.

## Current contribution scope
- Documentation clarifications
- Typo fixes
- Issue reports related to published materials

## Not currently accepting
- Core runtime changes
- Protocol-level modifications
- Feature requests without prior discussion

## Process
If you wish to contribute:
1. Open an issue describing the change or question
2. Wait for confirmation before submitting a pull request

Security issues should **not** be reported via GitHub issues.
See SECURITY.md for responsible disclosure instructions.
